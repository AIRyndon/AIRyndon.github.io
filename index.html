MSA Project

View the repo

Challenge

Create a Web based project using Typescript, React, Azure, and ASP.NET Core. This is required to pass phase two of the Microsoft Student Accelerator internship I attended on my first year’s summer break.

Response

I quickly read about the required tech-stack - I studied a bit of Typescript and React during those couple of weeks (luckily I already have some background with C# and JavaScript which made this a bit more manageable). I did all this during classes and it was intense - I remember this task being harder than all my assignments at the time.

Result

I was lucky that the MSA peeps was happy with my project - I got an internship and that same company hired me as a contractor after that internship. I remember being very happy when my teamate on that internship said I was technically skilled according to the MSA student partners :)

Connect Four

View the project

Challenge

Create a Web based Connect Four game. This is required to enter the Chingu team project.

Response

I used native JavaScript, some CSS and HTML for this project. As I was pressed for time while doing this, I used the ASP.NET Core boilerplate in Visual Studio. I believe that is a little overkill for a project of this scope, but I just wanted to get it done quickly.

Result

The moderators at Chingu were happy with my submission, it was simple and working as intended. The comment I received was to remove those console messages we developers use in our source code.
